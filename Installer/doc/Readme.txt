============================================
ChunkRootCopier - CHROCO (CHunk ROot COpier)
============================================

本プログラムは、コピー元ディレクトリから対象パスリストに記載された特定のファイルおよびディレクトリのみを抽出し、
コピー先ディレクトリに同じ構造を再現してコピーするためのツールです。
対象パスリストには、コピー元ディレクトリを基準とした相対パスを列挙します。
本プログラムはこれらのパスを検証し、
必要に応じてファイルシステム上の実際の大文字・小文字表記に合わせて補正したうえで順次コピーを実行します。
コピー先ディレクトリはデフォルトでクリアされ、空の状態から再構築されます。(オプションにより抑止可能)
各パスはディレクトリ階層ごとに作成され、
ファイルであればコピーされディレクトリであれば空ディレクトリとして作成されます。
本ツールは、プロジェクト内の必要な部分だけを抽出したい場合や、
配布用・バックアップ用に特定のファイル群のみを収集する用途に適しています。


----
インストール方法

Installer.exe を実行しインストールしてください。
